# MVP proves the manual core mechanic; AI/voice are deferred

The concept reads as "AI-driven behavioral orchestration," but the MVP ships with
**no AI, no voice/audio, no Sunday Review, no shadow metrics, and no push
notifications**. The MVP exists to validate one thing: that the core mechanic —
**Capacity → adapted Assignment → green Day** — is valuable on its own, built
entirely from manual user input. The AI onboarding, semantic Playbook editing,
audio-diary parsing, and the Iteration Engine are sequenced as later phases.

## Consequences

- A future contributor will find zero AI in a product described as AI-driven.
  That is deliberate, not an oversight — do not "fix" it.
- Life Areas are also deferred; the MVP uses a flat list of Goals.
- The MVP's value stands or falls on the manual mechanic alone.
